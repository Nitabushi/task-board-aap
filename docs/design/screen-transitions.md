## 画面遷移図（Mermaid記法）

```mermaid
flowchart TD
    START([アプリ起動 / URL直接アクセス])

    START --> AUTH_CHECK{認証済み？}

    AUTH_CHECK -- 未認証 --> S001[S-001: ログイン画面\n/login]
    AUTH_CHECK -- 認証済み --> S002[S-002: タスクボード画面\n/tasks]

    S001 -- ログイン成功 --> S002
    S001 -- ログイン失敗 --> S001_ERR[エラーメッセージ表示\n同画面に留まる]
    S001_ERR --> S001

    S002 -- ログアウトボタン --> S001
    S002 -- タスク追加ボタン --> S003_CREATE[S-003: タスク作成モーダル\n新規作成モード]
    S002 -- タスクカードクリック --> S003_EDIT[S-003: タスク編集モーダル\n編集モード]
    S002 -- D&Dでカラム移動 --> S002_STATUS[ステータス更新\nPATCH /api/tasks/:id/status]
    S002_STATUS --> S002

    S003_CREATE -- 保存 --> S002
    S003_CREATE -- キャンセル/✕ --> S002
    S003_EDIT -- 保存 --> S002
    S003_EDIT -- 削除 --> S003_CONFIRM[削除確認ダイアログ]
    S003_EDIT -- キャンセル/✕ --> S002
    S003_CONFIRM -- OK --> S002
    S003_CONFIRM -- キャンセル --> S003_EDIT

    S002 -- ユーザー管理\nadmin only --> ADMIN_CHECK{role = admin？}
    ADMIN_CHECK -- Yes --> S004[S-004: ユーザー管理画面\n/admin/users]
    ADMIN_CHECK -- No --> S002

    S004 -- ユーザー追加ボタン --> S005_CREATE[S-005: ユーザー作成モーダル\n新規作成モード]
    S004 -- 編集ボタン --> S005_EDIT[S-005: ユーザー編集モーダル\n編集モード]
    S004 -- 削除ボタン --> S004_CONFIRM[削除確認ダイアログ]
    S004 -- ヘッダーのTask Board --> S002
    S004 -- ログアウト --> S001

    S005_CREATE -- 保存 --> S004
    S005_CREATE -- キャンセル/✕ --> S004
    S005_EDIT -- 保存 --> S004
    S005_EDIT -- キャンセル/✕ --> S004
    S004_CONFIRM -- OK --> S004
    S004_CONFIRM -- キャンセル --> S004
```

## 遷移ルール一覧

### 認証ガード

| 遷移元         | 遷移先         | 条件                               |
| -------------- | -------------- | ---------------------------------- |
| 任意 → /login  | /tasks         | 認証済みの場合はリダイレクト       |
| 任意 → /tasks  | /login         | 未認証の場合はリダイレクト         |
| 任意 → /admin/users | /tasks   | 未認証 or role != 'admin' の場合   |

### ログイン

| 操作           | 遷移先             | 条件                             |
| -------------- | ------------------ | -------------------------------- |
| ログインボタン | /tasks             | 認証成功時                       |
| ログインボタン | S-001（エラー表示）| 認証失敗時                       |

### タスクボード

| 操作                 | 遷移先                     | 条件 |
| -------------------- | -------------------------- | ---- |
| ログアウトボタン     | /login                     | -    |
| タスク追加ボタン     | タスク作成モーダル（表示）  | -    |
| タスクカードクリック | タスク編集モーダル（表示）  | -    |
| D&Dでカラム移動      | 同画面（ステータス更新）    | -    |
| ユーザー管理リンク   | /admin/users               | admin のみ |

### タスクモーダル

| 操作             | 遷移先       | 条件         |
| ---------------- | ------------ | ------------ |
| 保存（成功）     | モーダルを閉じ、ボード更新 | バリデーション通過時 |
| 保存（失敗）     | モーダルを継続表示（フィールドにエラー表示） | バリデーションエラー時（422） |
| 削除ボタン       | 削除確認ダイアログ | 編集時のみ表示 |
| 削除確認 → OK    | モーダルを閉じ、タスク削除 | - |
| キャンセル / ✕  | モーダルを閉じる（変更破棄） | - |

### ユーザー管理

| 操作               | 遷移先                       | 条件               |
| ------------------ | ---------------------------- | ------------------ |
| ユーザー追加ボタン | ユーザー作成モーダル（表示）  | -                  |
| 編集ボタン         | ユーザー編集モーダル（表示）  | 有効ユーザーのみ   |
| 削除ボタン         | 削除確認ダイアログ           | 自分以外の有効ユーザー |
| 削除確認 → OK      | 同画面（論理削除後、一覧更新）| -                  |
